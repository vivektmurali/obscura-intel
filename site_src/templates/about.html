Why this exists

Portfolio evidence, not a trading signal.

This repo is portfolio evidence for UK data/statistics roles. The audience cares about method quality — pre-registration, null controls, multiple-testing correction, honest limitations — not Sharpe ratios. A well-executed null result is a full success for that purpose.

The discipline of committing a pre-registration before joining event data to price data, running a real permutation null instead of eyeballing significance, and reporting the result exactly as it came out — including a wrong-signed point estimate — demonstrates exactly the evaluation rigor those roles hire for.

The live pipeline that powers this site exists to prove the engineering runs unattended, not because the (null) result implies anything worth watching for. It has no server: everything here is static files rebuilt daily by a GitHub Actions cron and published to GitHub Pages.

Provenance

Every number on this site traces back to a cached raw API response and a committed, auditable pipeline. Source, decisions log, and full history: github.com/vivektmurali/obscura-intel.
